OliToki

HTML, CSS, JavaScript, bootstrap, branding, design

OliToki is a Korean-Fusion restaurant in the heart of Allston, MA. I was granted the opportunity to have full control over the branding, logo, and website development/design for this establishment. This work was done on a volunteer basis.

This project has been the most recent of my work and was also the first full web development project that I have completed for a client's business. They required a fully responsive design so customers will have full access on all devices when looking for restaurant hours, location, and menu. This website was intended to be primary information source for OliToki customers.
